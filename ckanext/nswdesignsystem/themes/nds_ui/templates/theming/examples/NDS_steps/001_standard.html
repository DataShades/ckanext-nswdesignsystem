Standard

Steps widget consist of a wrapper nds_steps and collection of multiple nds_step_item components. Items have content and title, and can be marked as filled with fill parameter.

By default, step items render title as a level-2 heading. Level can be changed via level parameter.

{%- call ui.util.call(ui.nds_steps) -%} {{ ui.nds_step_item(lipsum(1), title="First", fill=true) }} {{ ui.nds_step_item(lipsum(1), title="Second", level=4) }} {%- endcall %}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {%- call ui.util.call(ui.nds_steps) -%} {{ ui.nds_step_item(lipsum(1), title="First", fill=true) }} {{ ui.nds_step_item(lipsum(1), title="Second", level=4) }} {%- endcall %} {%- endraw %} {%- endcall %}
