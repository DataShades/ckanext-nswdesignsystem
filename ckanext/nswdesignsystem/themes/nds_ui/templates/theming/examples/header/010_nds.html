NSW Design system

Use submenu parameter of main_nav_item to render nested menus. The submenu is a list of items:

  • title: label shown to user
  • href: URL of the link
  • submenu: the same submenu structure for the nested menu level

{%- set submenu = [ {"title": "Simple", "href": "?simple"}, {"title": "Complex", "href": "?complex", "submenu": [ {"title": "Level 2", "href": "?level-2"} ]}, ] -%} {%- call ui.util.call(ui.header, title="Site title") -%} {%- call ui.util.call(ui.main_nav_wrapper) -%} {{ ui.main_nav_item("With submenu", href="", submenu=submenu, description=lipsum(1)) }} {{ ui.main_nav_item("No submenu", href="") }} {%- endcall %} {%- endcall %}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {%- set submenu = [ {"title": "Simple", "href": "?simple"}, {"title": "Complex", "href": "?complex", "submenu": [ {"title": "Level 2", "href": "?level-2"} ]}, ] -%} {%- call ui.util.call(ui.header, title="Site title") -%} {%- call ui.util.call(ui.main_nav_wrapper) -%} {{ ui.main_nav_item("With submenu", href="", submenu=submenu, description=lipsum(1)) }} {{ ui.main_nav_item("No submenu", href="") }} {%- endcall %} {%- endcall %} {%- endraw %} {%- endcall %}
