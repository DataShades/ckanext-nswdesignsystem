Standard

Global alert has optional title and cookie_name. When cookie_name specified, dismissed alert will be hidden for 24 hours.

{%- call ui.util.call(show_example) -%} {%- raw %} {{ ui.nds_global_alert(lipsum(1), title="Always shown") }} {{ ui.nds_global_alert(lipsum(1), title="Hidden for 24 hours", cookie_name="example_global_aler") }} {%- endraw %} {%- endcall %}

The variant parameter of global alert can be set to light or critical to change alert's appearance.

{%- call ui.util.call(show_example) -%} {%- raw %} {{ ui.nds_global_alert(lipsum(1), variant="light") }} {{ ui.nds_global_alert(lipsum(1), variant="critical") }} {%- endraw %} {%- endcall %}
