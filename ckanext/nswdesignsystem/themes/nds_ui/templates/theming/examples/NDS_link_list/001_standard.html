The nds_link_list component is a wrapper, that expects a series of nds_link_list_item components as a content.

{%- call ui.util.call(ui.nds_link_list) -%} {{ ui.nds_link_list_item("First", href="") }} {{ ui.nds_link_list_item("Second", href="") }} {{ ui.nds_link_list_item("Third", href="", blank=true) }} {%- endcall %}
