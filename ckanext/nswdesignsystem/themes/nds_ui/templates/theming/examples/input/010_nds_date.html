NSW Design System: Date input

A date input allows users to enter a date. It can be rendered as a group of fields using nds_date_input_group. To enable date picker, add parameter datepicker with value true.

{{ ui.nds_date_input_group("help text", label="Required field", name="date", required=true) }} {{ ui.nds_date_input_group(label="Invalid field", errors=["Invalid date"], value=["2020", "01", "10"]) }} {{ ui.nds_date_input_group(label="Datepicker", datepicker=true) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {{ ui.nds_date_input_group("help text", label="Required field", name="date", required=true) }} {{ ui.nds_date_input_group(label="Invalid field", errors=["Invalid date"], value=["2020", "01", "10"]) }} {{ ui.nds_date_input_group(label="Datepicker", datepicker=true) }} {%- endraw %} {%- endcall %}

There is also a single-field version of component - nds_date_input. Unlike the group, it always rendered with datepicker.

{{ ui.nds_date_input("help text", label="Required field", name="date", required=true) }} {{ ui.nds_date_input(label="Invalid field", errors=["Invalid date"]) }}
{%- call ui.util.call(ui.code, language="jinja2", dedent=true) -%} {%- raw %} {{ ui.nds_date_input("help text", label="Required field", name="date", required=true) }} {{ ui.nds_date_input(label="Invalid field", errors=["Invalid date"]) }} {%- endraw %} {%- endcall %}
