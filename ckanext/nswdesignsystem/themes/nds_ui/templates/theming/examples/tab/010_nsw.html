NSW Design System

The design system includes only interactive tabs that work in conjunction with tab panes. To render navigation tabs with links to other pages, do not use tabbed_content wrapper and instead add as_nav parameter to the tab_wrapper.

{%- call ui.util.call(show_example) -%} {%- raw %} {%- call ui.util.call(ui.tab_wrapper, as_nav=true) -%} {{ ui.tab("Link 1", href="") }} {{ ui.tab("Link 2", href="", active=true) }} {{ ui.tab("Link 3", href="") }} {%- endcall %} {%- endraw %} {%- endcall %}
